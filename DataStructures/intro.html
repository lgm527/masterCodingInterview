    How Computers Store Data

  • CPU: does actual work/computing, also has a very small cache of very recent info that was accessed.
  • RAM (random access memory): loose memory when computer shuts down, can be accessed faster. Contains collection of addresses. 8 bits = byte
  • Storage (video, pictures, music, files): long term storage
